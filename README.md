This project analyzes Instagram activity data of a tech influencer to uncover engagement trends, reach distribution, and follower growth insights.

The analysis was completed as part of the **Codebasics Internship Challenge**.

---

## Project Objectives

- Analyze content performance
- Identify engagement patterns
- Track follower growth trends
- Generate insights to improve content strategy

---

## Dataset Overview

The project contains three tables:

### fact_content
Contains post level metrics:
- impressions
- reach
- likes
- comments
- shares
- saves

### fact_account
Contains account level metrics:
- profile_visits
- new_followers

### dim_dates
Contains date attributes:
- month_name
- weekday_name
- weekday_or_weekend
- week_no

---

## Key Insights

- Reels generated the highest reach across all post types
- Certain tech categories drove significantly higher engagement
- Viral posts contributed strongly to follower growth spikes
